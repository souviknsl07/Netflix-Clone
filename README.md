## Prerequisites
- Have knowledge about Reactjs.
- Have knowledge about REST API's.

### Follow the steps below to install it on your local machine.

## Installation Steps

### Using npm

Run commands

1. `npm install`

2. `npm start`

### Or using yarn

Run commands

1. `npm install --global yarn`

2. `yarn install`

3. `yarn start`

It will start the development server at `localhost:3000`
